# 部署指南

## 第一步：开通云开发环境

### 1.1 开通云开发

1. 打开微信开发者工具，导入 `fishing-buddy-miniprogram/` 目录
2. 点击工具栏「云开发」按钮（云朵图标）
3. 弹窗中点击「开通」
4. 选择免费的基础版（0 元/月）
5. 等待环境创建完成（约 30 秒）

### 1.2 创建数据库集合

在云开发控制台 → 数据库：

1. 点击「+」创建集合，名称输入 `trips`
2. 权限设置为「仅创建者可读写」
3. 再点击「+」创建集合，名称输入 `settings`
4. 权限设置为「仅创建者可读写」

### 1.3 部署云函数

在微信开发者工具的文件目录中：

1. 右键点击 `cloudfunctions/weather-proxy/` → 选择「上传并部署：云端安装依赖」
2. 等待部署完成（约 10 秒）
3. 右键点击 `cloudfunctions/reverse-geocode/` → 选择「上传并部署：云端安装依赖」
4. 等待部署完成

### 1.4 验证

重新编译小程序，首页应该不再报云函数调用错误（天气数据可能为空，因为还没配 API Key，属正常）。

## 第二步：注册 API 并配置密钥

### 2.1 OpenWeatherMap

1. 打开 https://openweathermap.org/api
2. 点击「Sign Up」注册账号（免费）
3. 登录后点击「API keys」标签页
4. 复制你的 API Key（默认已有 `Default` key，等 2 小时后生效）
5. 免费额度：1000 次/天，足够个人使用

### 2.2 腾讯地图

1. 打开 https://lbs.qq.com/dev/console/application/mine
2. 用微信扫码登录
3. 点击「创建应用」→ 应用名称填「渔游记」→ 应用类型选「小程序」
4. 点击应用下的「添加 key」→ 勾选「WebServiceAPI」
5. 提交后复制生成的 Key

### 2.3 配置云函数环境变量

在微信开发者工具的文件目录中：

**天气代理：**

1. 右键 `cloudfunctions/weather-proxy/` →「云函数详情」
2. 点击「编辑」→ 环境变量 → 添加
3. 名称：`OPENWEATHERMAP_API_KEY`
4. 值：你复制的 OpenWeatherMap Key
5. 保存并重新部署

**反向编码：**

1. 右键 `cloudfunctions/reverse-geocode/` →「云函数详情」
2. 点击「编辑」→ 环境变量 → 添加
3. 名称：`TENCENT_MAP_KEY`
4. 值：你复制的腾讯地图 Key
5. 保存并重新部署

### 2.4 验证

重新编译小程序，首页应显示真实天气数据和钓鱼指数评分。记录行程时点击「获取定位」应返回真实地名。

## 第三步：提交审核发布

### 3.1 填写小程序信息

登录 https://mp.weixin.qq.com → 设置 → 基本设置：

- 小程序名称：渔游记
- 小程序头像：上传一张钓鱼相关图片（建议 144×144px）
- 小程序介绍：钓鱼指数助手，查看每日钓鱼指数，记录钓鱼行程
- 服务类目：工具 → 信息查询

### 3.2 上传代码

在微信开发者工具中：

1. 点击右上角「上传」按钮
2. 版本号填 `1.0.0`
3. 备注填「首发版本：钓鱼指数 + 行程记录 + 历史查看」
4. 点击上传

### 3.3 提交审核

1. 登录 https://mp.weixin.qq.com → 版本管理
2. 在「开发版本」中找到刚上传的代码
3. 点击「提交审核」
4. 填写审核信息（功能页面说明、测试账号等）
5. 提交

### 3.4 发布

审核通过后（通常 1-3 个工作日）：

1. 登录 https://mp.weixin.qq.com → 版本管理
2. 点击「发布」
3. 用户通过微信搜索「渔游记」或扫码即可使用
