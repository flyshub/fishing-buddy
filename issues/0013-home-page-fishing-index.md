## Parent

#0010 微信小程序迁移

## What to build

实现小程序首页，展示今日钓鱼指数大卡片、7 天预报列表、最近 3 条行程摘要。首页是核心价值入口，用户打开小程序第一眼即见钓鱼指数。

## Acceptance criteria

- [ ] 首页加载时调用天气代理云函数获取当前定位的 7 天预报
- [ ] 今日钓鱼指数大卡片显示：总分、推荐等级、8 因子得分 Chip、今日天气详情
- [ ] 7 天预报列表每行显示：日期、天气状况、气温范围、指数评分、推荐等级
- [ ] 每行可展开查看 8 因子详细得分（进度条 + 分数）
- [ ] 最近 3 条行程摘要（钓点、时间、鱼种、数量、空军标记）
- [ ] 天气数据加载中显示 loading 状态
- [ ] 天气数据获取失败显示错误提示 + 重试按钮
- [ ] 无行程数据时显示空状态引导
