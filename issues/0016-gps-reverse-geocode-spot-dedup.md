## What to build

在记录行程流程中集成 GPS 定位能力：获取用户经纬度、反向地理编码自动填入钓点名、Haversine 公式匹配 100m 内历史钓点。用户点击"获取定位"按钮触发（非自动），避免授权弹窗影响体验。

## Acceptance criteria

- [ ] 记录页面提供"获取定位"按钮，点击后调用 wx.getLocation
- [ ] 用户拒绝授权时显示友好提示，不崩溃
- [ ] 获取经纬度后调用反向编码云函数，自动填入钓点名输入框
- [ ] 反向编码云函数调用腾讯地图 WebService API，API Key 存环境变量
- [ ] 反向编码失败时钓点名输入框保持空白，用户可手填
- [ ] 获取定位后查询云数据库 trips 集合，用 Haversine 公式检查 100m 内是否有历史钓点
- [ ] 匹配到历史钓点时，自动使用已有钓点名，并提示用户"已匹配历史钓点：XXX"
- [ ] 未匹配到历史钓点时，使用反向编码结果填入
- [ ] Haversine 距离计算单元测试通过
