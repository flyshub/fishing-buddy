## What to build

实现记录行程页面，用户填写表单后将行程数据以嵌套文档结构写入云数据库 trips 集合。支持鱼种动态添加/删除、日期时间分别选择、空军记录。

## Acceptance criteria

- [ ] 表单包含：出发时间（日期+时间选择器）、钓点名（手填）、装备类型、鱼种列表（动态添加/删除）、饵料、最大长度、总重量、备注
- [ ] 鱼种从内置 80-100 种列表中选择（picker），支持搜索
- [ ] 饵料从内置列表中选择，支持自定义输入
- [ ] 鱼种数量支持动态添加/删除，至少保留 1 行
- [ ] 提交时校验：至少填写 1 种鱼种且数量 > 0，否则提示错误
- [ ] 提交成功写入云数据库 trips 集合，嵌套文档结构（species 数组、weather 对象、spot 嵌入）
- [ ] 提交成功后表单重置，显示成功提示
- [ ] 提交失败显示错误提示
- [ ] 支持空军记录（不填鱼种时跳过校验，species 数组为空）
- [ ] 日期选择器限制不早于 2000 年，不晚于明天
